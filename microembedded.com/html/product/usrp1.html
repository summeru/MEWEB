USRP1

编号：086-782748-01

询问报价

USRP1-PKG 母板套件包括：USRP 母板，外壳，2 个 RF 电缆，USB 电缆，直流电源，及安装附件

1.产品介绍:

       USRP1是原通用软件无线电外设™硬件（USRP），提供入门级的射频处理能力。它的目的是为成本敏感的用户和应用程序提供软件定义无线电开发能力。该架构包括Altera公司的Cyclone FPGA，64 MS / s的双通道ADC，128 MS / s的双通道DAC和USB2.0连接提供数据到主机处理器。模块化设计允许USRP1到运营从DC到6GHz。该平台USRP1可以支持两个完整的RF子板。这一特性使得USRP适用于需要发射和接收链，或双频双发射的高隔离应用程序/接收操作。该USRP1可传送多达8 MS / s的和从主机应用程序，用户可以在FPGA架构中实现自定义的功能。

2.特性 - FEATURES:

  • 采样的量化精度为 8 bit 情况下射频带宽（RF bandwidth）为 16 MHz
  • 采样的量化精度为 16 bit 情况下射频带宽（RF bandwidth）为 8 MHz
  • USB 2.0 高速连接
  • 低成本解决方案
  • MIMO 可构建性 - USRP 母板上的两路收发使得单个 USRP 可以构建 MIMO
  • FPGA：Altera Cyclone
  • ADCs: 12-位 64 MS/s
  • DACs: 14-位 128 MS/s

3.产品包含:

USRP1-PKG 母板套件包括：USRP 母板，外壳，2 个 RF 电缆，USB 电缆，直流电源，及安装附件。
