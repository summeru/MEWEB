USRP X300

编号:086-783144-01

询问报价

KINTEX7-325T FPGA，2个通道，万兆以太网和PCIe总线

1.产品介绍:

       USRP X300是一种高性能、可扩展的软件定义无线电（SDR）的平台，可用来设计和部署下一代无线通信系统。其硬件架构结合了覆盖DC - 6 GHz甚至高达120MHz基带带宽的、多个高速接口选项（PCIe，双10千兆位以太网，双1千兆以太网）的两个扩展带宽的子板槽，和可安装在方便的桌面或者机架安装的半宽1U外形的一个大的用户可编程的Kintex-7 FPGA。除了提供同级别最佳的硬件性能，X300的开源软件架构提供了跨平台的UHD驱动程序的支持使得它兼容大批支持开发框架，参考架构和开源项目。

2.特性 - FEATURES:

  • 两个宽频带RF子板插槽
    • 每个高达120MHz的带宽（宽带版本CBX，WBX，SBX）
    • 子板可选择覆盖直流至6 GHz
  • 为高性能DSP(XC7K325T)提供大量可定制Xilinx FPGA Kintex-7多个高速接口
    • 双10千兆位以太网 - 200 MS / s的全双工
    • 快速PCIe（桌面）- 200 MS / s的全双工，10 us延迟
    • ExpressCard（笔记本）- 50 MS / s的全双工，10 us延迟
    • 双千兆以太网 - 25 MS / s的全双工
  • UHD架构提供了与下列的兼容性
    • GNU Radio
    • C++/Python API
    • Amarisoft LTE 100
    • 其他的第三方软件和框架
  • 灵活的时钟架构
    • 可配置的采样时钟
    • 可选的GPS训导的OCXO
  • 紧凑和坚固的半宽1U外形安装，使用于方便的桌面或机架
  • 数字I / O面板可访问用户控制和FPGA上的接口上

3.产品包含:

  • USRP X300
  • 1个千兆以太网电缆
  • 1千兆以太网的SFP适配器
  • 电源及美国线
  • 内部JTAG适配器的USB 2.0电缆
  • 4个SMA隔板电缆
  • 入门指南
